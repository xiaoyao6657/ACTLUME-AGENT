# Changelog

## 0.1.0

- Initial local ReAct agent CLI.
- Added project scanning, project-summary caching, and context indexing.
- Added edit planning, change summaries, automatic checks, and limited repair loops.

- Added interactive CLI history, multiline input, colored output, helper commands, and benchmark coverage.
- Added CI workflow, unit tests, benchmark suite, and npm dry-run release script.
